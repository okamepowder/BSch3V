1. 概要

  BSch3V (Basic Schematic 3V) はWindows 10/11で(そしてたぶん7以降で)動作するフリーソフトウェアの回路図エディタです。 操作の見通しが悪くならないように、一般的な回路図作成に必要な基本的な機能に絞ってあります。
  本パッケージは、BSch3Vとその周辺ユーティリティソフトウェアをまとめたものです。
  
  BSch3Vとその周辺ユーティリティソフトウェアは、Version 0.8x から、UNICODEを扱うようになりました。保存ファイルの文字コードはUTF-8になります。
  (Version 0.7xまでのシフトJISのファイルはそのまま読み込むことができます)

3. インストール  

1. 準備   ランタイムライブラリのインストール
  BSch3Vシリーズをお使いになる前に、ランタイムライブラリである「 Microsoft Visual C++ 2015-2022 再頒布可能パッケージ (x86)」を予めインストールしておく必要があります。
  Runtimeフォルダに、ランタイムライブラリのインストールプログラム VC_redist.x86.exe を収めています。これを実行すると、ランタイムライブラリのインストールができます。  
  旧パッケージとはBSch3Vとはビルド環境が変わりました。これまで、BSch3V Version 0.83.02とそれより古いものをお使いの場合は、ランタイムライブラリのインストールが必要な場合があります。

2. BSch3Vパッケージ版のインストール
  BSch3V自体には特にインストーラは準備していません。

  • 初めてBSch3Vをお使いの方や、これまでBSch3Vをお使いになっていなかったPCでお使いになる場合は…
      解凍したパッケージ版のフォルダの階層構造そのまままるごとユーザーフォルダの下の適当なフォルダに移動（またはコピー)してください。ユーザーフォルダ外("C:\Program files"など)に保存することはお奨めできません。
      あとは必要に応じて、exeファイルのショートカットを作ってください。
  • お使いのPCでこれまでもBSch3Vをお使いになっていた場合は…
      解凍したパッケージ版の中のLIBフォルダ、BSCH3.INI以外を、現在お使いになっているBSch3Vのフォルダに移動（またはコピー)してください。

  ユーザーインターフェースが英語になる場合は、exeファイルと同じフォルダに、一緒に配布された *_lang.dllがあるかどうか確認してください。

5. Version 0.8x以降の固有の注意点(Version 0.7x以前をお使いの方へ)

  • ファイルの文字コード
    BSch3Vシリーズ Version 0.8x以降が読み書きするファイルの文字コードはBOMなしのUTF-8です。
  • Version 0.7xまでのファイルの読み込みについて
    BSch3V Version 0.7xまでのファイルについては、そのまま読めます。
    Qt-BSch3Vで作成したファイルは、シフトJIS以外の文字コードの場合があります。その場合は文字コード変換ソフトなどを使って、いったんシフトJISに変換してください。
  • BSch3V Version 0.8x以降で保存したファイルを0.7xまでで読めるようにするには
    文字コード変換ソフトでUTF-8をシフトJISに変換すると読み込めます。
    ただし、変換したファイルを0.7x以前で読み込もうとすると、「このファイルは新しいバージョンで…」というメッセージが現れます。残念ですが、0.80以降に追加された機能を使っているデータは失われます。
  • 部品リスト作成ソフト PL3Wの出力ファイル
    Version 0.7xまでは、ローカルエンコーディングのCSVとSLKファイルでした。
    Version 0.80.00 では、UTF-8のBOMコード付きのCSVと、XMLファイルです。

    CSVファイルは、先頭にBOMコードが付いていないと Excelがローカルエンコード(シフトJIS)と見做してしまい、PL3Wが出力したデータは文字化けします。
    BOMコードを付けることで、ExcelがUTF-8と認識するようにしています。
    自作プログラムでPL3Wが出力したCSVを取り扱われる場合は、先頭の3バイトのBOMコード(EF BB BF)を意識するか、文字コード変換ソフトを使ってシフトJISに変換してください。
    なお、Excel 2007以降で読み込まれるのであれば、XMLファイル形式の方が読み込み時の情報欠落は少ないはずです。

    水魚堂オンラインでは、.NET版のPL3Wを配布しています。こちらは XLSXフォーマットで出力しますので使い勝手はいいかもしれません。
  • 旧BSchのデータのコンバータ CE2TO3について
    Version 0.7xまでのパッケージに含まれている、CE2TO3 がそのまま使えます。
    CE2TO3は、ローカルエンコード(日本語環境ではシフトJIS)のファイルを生成しますが Version 0.8xは、ローカルエンコードのファイルをそのまま読み込むことができます。
